# Beer-o-Meter
Small vibe-coded app for tracking amount of beer drank by each group (in this project by dormitory floors).

Cast votes at [127.0.0.1:5001/vote_page](127.0.0.1:5001/vote_page), see live results at [127.0.0.1:5001](127.0.0.1:5001). You can also find these pages on your LAN IP address.

## Setup
```sh
pip install -r requirements.txt`
python3 server.py`
```
And you are live! Reset votes by editing [votes.json](votes.json).
